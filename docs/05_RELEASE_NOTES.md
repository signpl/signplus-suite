# Release Notes

사용자에게 전달되는 업데이트 내역입니다. 개발 변경 이력(내부용)은 `04_CHANGELOG.md`를 참고하세요.

## v3.8.6

- 베타 테스터 전원이 사용할 수 있는 공용 체험판 시리얼을 추가했습니다.
- 라이선스 입력창에서 다양한 형식의 시리얼 번호를 잘림 없이 입력할 수 있도록 개선했습니다.

## v3.8.5 — Trial / Pro 라이선스

- 라이선스 체계를 Trial(체험판, 30일)과 Pro(정식, 무제한) 두 가지로 정리했습니다.
- 사이드바와 관리자 화면에서 현재 라이선스 종류(Trial Version / Pro Version)를 명확히 표시합니다.

## v3.8.3 — 프로젝트 상태 통합

- 견적, 프로젝트, 대시보드가 하나의 공통 상태 기준(상담중/견적발송/계약/시공중/완료)을 사용하도록 통일했습니다.
- 프로젝트 상세 화면에서 상태를 바로 변경할 수 있으며, 변경 즉시 연결된 견적과 대시보드에 반영됩니다.
- 기존에 저장된 견적/프로젝트 데이터는 그대로 이용할 수 있습니다.

## v3.8.0 — 관리자 라이선스 대시보드

- 관리자용 라이선스 관리 화면을 추가했습니다.
- 프로젝트 대시보드의 상태 연동과 화면 구성을 개선했습니다.

## v3.7.2 — 프로젝트-견적 연동 안정화

- 프로젝트 카드에 연결된 견적의 상태와 금액이 항상 정확히 표시되도록 수정했습니다.
- 프로젝트 카드 요약 표시를 "견적 N건 / 총금액 / 상태" 형식으로 보기 쉽게 바꿨습니다.

## v3.7.0 — 대시보드 실시간 갱신

- 견적을 저장하면 프로젝트 대시보드가 자동으로 최신 정보를 반영하도록 했습니다.
- 프로젝트 추가 시 저장된 견적을 선택해 거래처·프로젝트명·금액을 자동으로 채울 수 있습니다.
