# Signplus Suite Vision

Signplus Suite is not only a signage quotation program.

It is being developed as a complete platform for the Korean signage and advertising industry.

Long-term Goals

- Desktop ERP
- Cloud Platform
- Supplier Network
- Community
- Marketplace
- AI Assistant
- Advertisement Platform
- Chat
- Joint Purchasing

==================================================

# Development Principles

- Offline-first architecture.
- Cloud-ready architecture.
- Business logic must be independent from storage.
- Preserve backward compatibility.
- Never remove existing features without approval.
- SQLite remains the primary local database.
- Future NAS/Cloud support must require minimal code changes.

# Core Philosophy

The desktop application must always work offline.

Cloud features are optional enhancements.

The desktop application must never depend on an internet connection.
